# Nautilus-Konsole
Just a very simple extension to open a directory in Konsole using Nautilus.

# Installation
This extension is available in AUR for those using ArchLinux as [nautilus-konsole](https://aur.archlinux.org/packages/nautilus-konsole).

Otherwise, clone the git repo and run the `install.sh` script that just basically copies a file.
```
git clone https://github.com/abdurehman4/nautilus-konsole.git
cd nautilus-konsole
./install.sh
```
